Types of tags in HTML


There are two types of tags in html
  • Container tags
  • Empty tags

CONTAINER TAGS

Conatiner tags are the tags that contain both on and off tags.
For eg.- The tags such as < body> < html> are container tags and their closing tags follow respectively

EMPTY TAGS

Empty tags are the tags that only contain the on tag.
For eg.- The tags such as < br> < P> do not have anyy closing tags
